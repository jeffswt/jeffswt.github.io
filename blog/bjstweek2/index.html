再说一下这周的情况吧:

  1. 本周做的题目数量较少, 题目类别包含以下几类:
  2. 图论: 好多好多最小割, 最短路优化的本来应该最小割的对偶图, 动态建边不停 TLE 的 SPFA 费用流;
  3. 数据结构: Query 写错怒调一下午的区间 Splay, 不带路径压缩的瞎搞并查集, Rotate 写错怒调一整天的老梗 LCT (弹飞绵羊, 不必多说, 还是比较重要的);
  4. 动态规划: 动态规划, 斜率优化动态规划;
  5. 数论与矩阵: 拉格朗日乘数原理 (虽然还是没有弄懂), 用 Python 水的高精度, 快速幂矩阵乘法, 矩阵大小步算法, 还有想都想不通的欧拉函数;
  6. 计算几何: 没开 long long 见祖宗的扫描线巨水 STL, 奇葩暴搜, 双重平面凸包;
  7. 差不多就是这些了, 不算太多. 还有一些应该做的题记在这里: 7 道 QTREE 和 7 道 GSS 没有做, 加上应该不成问题的 poj3580 来做一遍~
